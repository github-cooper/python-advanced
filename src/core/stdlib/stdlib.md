# Networking and Interprocess Communication
- 网络和进程间通信
- 模块提供了网络和进程间通信的机制
- 某些模块仅适用于同一台机器上的两个进程，例如 signal 和 mmap 。 其他模块支持两个或多个进程可用于跨机器通信的网络协议。
